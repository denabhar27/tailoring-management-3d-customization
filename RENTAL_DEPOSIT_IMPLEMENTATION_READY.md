## 💰 Payment Flow

### Before (Old System)
```
Downpayment = Price × 50%
```

### After (New System)
```
Due on Pickup = (Price × 50%) + Full Deposit
Balance on Return = Price × 50%
```

### Example
```
Item: Suit
Price: ₱500
Deposit: ₱1000

Due on Pickup: ₱1250 (₱250 rental + ₱1000 deposit)
Balance on Return: ₱250 (remaining rental)
```
